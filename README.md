# Web Application Security - Task 3

To understand and analyze common web application vulnerabilities based on OWASP Top 10 and learn mitigation techniques.

## Vulnerabilities Covered
1. SQL Injection
2. Cross Site Scripting (XSS)
3. Cross Site Request Forgery (CSRF)
4. File Inclusion
5. Burp Suite Request Analysis
6. Security Headers Analysis

- Primebook 2 Pro
- Google Chrome
- GitHub
- Security Headers Analyzer

## Outcome
Successfully studied common web application vulnerabilities and their prevention techniques.
